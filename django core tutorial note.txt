DIFFERENCE BETWEEN git switch -c dev and git checkout -b dev

they only achieve the same thing in term of branches alone,git switch was introduce for the purpose of
Safety:

	- git switch is safer
	- It prevents accidental file overwrites
	- Git introduced it to reduce beginner mistakes

this is what git checkout -b dev does:

	- git checkout can:
	- switch branches
	- create branches
	- restore files
	- detach HEAD
	- overwrite working files
	
git switch only handles branches ‚Äî nothing else.
